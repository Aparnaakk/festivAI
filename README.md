# festivAI
Customized and Personal greeting cards using artificial intelligence and augmented reality
Currently, our project covers three festivals:
  1. Diwali
  
  2. Halloween 
  
  3. Valentines day
  
  PREREQUISITES:
  
    1. Tensorflow
    
    2. Numpy
    
    3. Scipy
    
    4. Pandas
